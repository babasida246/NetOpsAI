# Changelog

All notable changes to NetOpsAI IT Gateway will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [6.0.0] - 2026-01-30

### Added
- Complete setup wizard for first-time system configuration
- Multi-language support (Vietnamese and English)
- AI-powered chat interface with multiple provider support
- Asset management system (QLTS)
- CMDB (Configuration Management Database)
- Warehouse and spare parts management
- Purchase planning and asset increase workflows
- Docker deployment configuration
- PostgreSQL database with comprehensive schema
- Redis caching layer
- MCP (Model Context Protocol) server integration
- RESTful API with OpenAPI specification
- User authentication and session management
- Role-based access control
- System health monitoring
- Logging and observability

### Changed
- Migrated from default admin to setup wizard
- Improved Docker build process
- Enhanced security configurations
- Optimized database initialization

### Fixed
- Setup page authentication flow
- Database reset functionality
- Accessibility warnings in forms
- Docker container dependencies
- Build process for all packages
