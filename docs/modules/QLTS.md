## Integration

### With Assets Module

When asset increase is confirmed:
1. Assets are automatically created in Assets module
2. Asset codes are generated
3. Location assignments are applied
4. Depreciation schedules start

### With CMDB Module

For IT assets:
1. CIs can be auto-created
2. Relationships are established
3. Service mappings are applied

## Best Practices

1. **Complete Planning**: Include all required items
2. **Accurate Estimates**: Research current prices
3. **Timely Submission**: Submit plans on schedule
4. **Serial Numbers**: Record for each unique item
5. **Document Attachments**: Upload invoices, quotes
